# mytts

A Python library for generating narrated audio and videos from structured scripts.

## ✨ Features

- 🎙️ Text-to-speech using Microsoft Edge TTS
- 🧠 Script parser with:
  - slide structure
  - inline pauses
  - voice control
- 🔊 Audio enhancement (filters + normalization)
- 🎬 Video generation with ffmpeg
- 📝 Optional subtitle generation (.srt)
- 🖼️ PDF → slide images
